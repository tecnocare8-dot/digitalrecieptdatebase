## 1. アプリ概要
紙のレシートをスマートフォンのカメラで撮影し、デジタルデータとして保存・管理するアプリケーションです。
インボイス登録番号（T+13桁）から企業名を自動検索する機能も備えています。

## 2. 主な機能と使い方

### 2.1 レシートをスキャンする（トップページ）
1.  **「カメラ起動」**ボタンを押し、レシートを撮影します。
2.  撮影された画像が表示されます。必要に応じて**「解析実行」**ボタンを押すと、OCR（文字認識）により、「日付」「合計金額」「登録番号」などを自動で読み取ります。
3.  読み取った内容を確認・修正し、**「保存」**ボタンを押してください。

### 2.2 履歴を確認・検索する（履歴ページ）
保存したレシートの一覧を確認できます。

*   **基本表示**: 新しい日付順に一覧表示されます。
*   **詳細検索**: 画面上部の「🔍 詳細検索・フィルタリング」ボタンを押すと、条件を指定できます。
    *   **日付範囲**: 「いつから」「いつまで」を指定。
    *   **金額範囲**: 「いくら以上」「いくら以下」を指定。
    *   **キーワード**: 会社名や備考欄の内容で検索。
    *   **削除済みを含む**: チェックを入れると、過去に削除したレシートも検索結果に表示されます（監査用）。

### 2.3 詳細確認・編集・履歴（詳細画面）
一覧の「画像・詳細」「編集」ボタンから開きます。

*   **画像の拡大**: マウスホイールやピンチ操作で画像を拡大・移動できます。
*   **内容の編集**: 日付や金額などを修正し「保存する」で更新します。
*   **変更履歴（監査ログ）**: 編集画面の下部に、過去の変更履歴（いつ・誰が・何を修正したか）が表示されます。

### 2.4 CSVダウンロード
履歴ページ右上の「CSVダウンロード」ボタンから、現在表示されているデータをExcel等で開けるCSV形式でダウンロードできます。

## 3. よくある質問 (FAQ)

*   **Q: 削除したレシートは完全に消えますか？**
    *   A: いいえ。誤操作防止と法的要件のため、「論理削除」としてデータは残ります。「詳細検索」で「削除済みを含む」にチェックを入れると確認できます。
*   **Q: 企業名が自動で出ません。**
    *   A: インボイス登録番号（T+数字13桁）が正しく読み取れているか確認してください。国税庁APIに登録がない企業や、番号がないレシートの場合は手動で入力してください。
